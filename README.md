# psr-log

## Description

Common Log Interface ([PHP FIG][2] [PSR-3][3]).

## Requirements

 * node: 10

## Installation

Through [NPM](https://www.npmjs.com) as [@chubbyjs/psr-log][1].

```sh
npm i @chubbyjs/psr-log@1.0.2
```
